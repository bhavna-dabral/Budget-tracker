# 💰 MERN Expense Tracker

A modern **full-stack Expense Tracker** built with the **MERN Stack (MongoDB, Express.js, React.js, Node.js)** to help users manage personal finances efficiently.

Track your **income**, **expenses**, **balance**, visualize spending trends with **interactive charts**, and get **smart financial insights** through a responsive dashboard.

## ✨ Features

### 🔐 Authentication System
- User Signup & Login
- Protected Dashboard Routes
- JWT Token Authentication
- Logout Functionality
- Reset Password Flow

### 💵 Income & Expense Management
- Add Income Transactions
- Add Expense Transactions
- Delete Transactions
- Categorized Records
- Real-time Total Balance

### 📊 Analytics Dashboard
- Total Income / Expense / Balance Cards
- Monthly & Yearly Reports
- Interactive Charts (Chart.js)
- Budget Usage Progress Bar
- Expense Prediction

### 🧠 Smart Insights
- Overspending Alerts
- Highest Spending Category
- Monthly Comparison
- Savings Rate Insights

### 📄 Utilities
- Export Reports to PDF
- Mobile Responsive UI
- Clean Modern Dashboard Design
